# Lab 1 - Cadastro de Aluno

Este é o formulário de cadastro de aluno para o Lab 1.

## O que eu fiz:
* **Labels:** Usei o `for` para ligar as legendas aos campos de texto.
* **CPF:** Coloquei um padrão (pattern) para o formato 000.000.000-00 e fiz o teclado numérico abrir no celular.
* **Acessibilidade:** Usei o `aria-describedby` no CPF para mostrar a dica de formato para quem usa leitor de tela.
* **Organização:** Usei `fieldset` e `legend` para separar as opções de "Turno".
* **Validação:** Usei `type="email"` e o atributo `required` para obrigar o preenchimento dos campos.
